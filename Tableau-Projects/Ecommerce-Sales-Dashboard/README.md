# Ecommerce Sales Analysis Dashboard

## Tools & Technologies Used
- Tableau
- Microsoft Excel
- Interactive Dashboard Design
- Data Visualization Techniques

## Project Overview
This Tableau dashboard analyzes ecommerce sales performance by tracking sales trends, customer behavior, product performance, payment methods, and country-level sales insights.

## Key Business Objectives
- Analyze total sales, orders, and quantity sold
- Identify top-performing product categories
- Compare sales across countries
- Track order status performance
- Identify top customers and top products

## Dashboard Features
- KPI cards for Total Sales, Total Orders, Total Quantity, and Average Order Value
- Monthly sales trend analysis
- Category-wise sales breakdown
- Country-wise sales analysis
- Payment type distribution
- Top N products by sales
- Brand-wise quantity analysis
- Customer performance dashboard

## Insights Generated
- Identified top-selling product categories
- Highlighted highest revenue-generating countries
- Tracked order completion and return status
- Analyzed customer purchase contribution
