# Pizza Sales Dashboard Project

## Tools & Technologies Used
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Data Visualization Techniques

## Project Overview
This Power BI dashboard project provides a detailed analysis of pizza sales performance by examining 
revenue trends, order patterns, product categories, and customer preferences. The dashboard is designed to 
help understand business performance through interactive filters and visual insights.

## Key Business Objectives
- Analyze total revenue generated from pizza sales
- Track order volume across different time periods
- Identify top-performing pizza categories and sizes
- Monitor monthly revenue trends
- Evaluate top revenue-generating pizza products

## Dashboard Features
- Interactive slicers for Pizza Name, Pizza Size, and Pizza Category
- Revenue trend analysis by month
- Order distribution by hour and day
- Category-wise revenue contribution
- Top 5 pizzas by revenue
- Running total revenue analysis
- MTD (Month-to-Date), YTD (Year-to-Date), and QTD (Quarter-to-Date) performance tracking

## Insights Generated
- Identified highest revenue-generating pizza categories
- Tracked peak order hours for customer demand analysis
- Compared monthly and quarterly business performance
- Highlighted product contribution to total revenue
