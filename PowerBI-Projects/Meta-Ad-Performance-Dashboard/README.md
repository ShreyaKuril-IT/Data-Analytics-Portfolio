# Meta Ad Performance Dashboard

## Project Overview
This Power BI dashboard analyzes Meta advertising campaign performance across Facebook and Instagram platforms.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel/CSV

## Key KPIs
- Impressions
- Clicks
- Shares
- Comments
- Engagement Rate
- CTR
- Conversion Rate
- Purchases

## Dashboard Pages
- Audience Analysis
- Performance Analysis
- Campaign Insights
- AI Insights
